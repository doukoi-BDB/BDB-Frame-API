>  ### 使用说明 ：
>
> 一、使用 composer 拉取对应版本代码
> 
> 1）拉取特定版本1.0.1 ： composer require txz/sdk:1.0.1
> 
> 2）拉取项目的5.5系列的版本：composer require "txz/sdk:5.5.*"
> 
> 3）如没有composer ，可以的话，官方装一个非常快，如果懒得装，那么常规 git 下载安装包即可。
>
